Stage de Deuxième Année

Description du Stage

Durant mon stage de deuxième année, j'ai développé deux projets distincts : une application de gestion pour un garagiste et un site web de vente de véhicules.

1. Application de Gestion

Une application permettant au garagiste de gérer son stock de véhicules et leurs modifications :

  • Saisie des véhicules achetés avec leurs prix d'achat
  • Gestion des modifications apportées aux véhicules
  • Calcul automatique des coûts totaux
  • Aide à la fixation des prix de vente

Captures d'écran de l'Application

2. Site Web MyCarDeals

Une plateforme web complète pour la vente de véhicules, comprenant un site public et une interface d'administration.

Architecture du Site Web

Le projet est structuré en trois parties principales :

  • Frontend : Interface utilisateur publique et administration
  • Backend : API Express.js
  • Base de données : Firebase (Firestore)

Technologies Utilisées

  • Node.js avec Express
  • JavaScript
  • HTML5
  • CSS3
  • Firebase
  • Firestore

Fonctionnalités du Site Web

Site Public

  • Page d'accueil avec véhicules en vedette
  • Liste des véhicules avec filtres
  • Pages de détails des véhicules
  • Système de recherche

Interface d'Administration

  • Tableau de bord avec statistiques
  • Gestion des véhicules (CRUD)
  • Authentification sécurisée
  • Gestion des équipements

Structure des Données

La base de données utilise trois collections principales :

  • vehicles : Données normalisées des véhicules
  • cars : Format historique des véhicules
  • admins : Gestion des administrateurs

Configuration Firebase

Le projet utilise plusieurs fichiers de configuration Firebase :

  • firebase.json : Configure le déploiement et les services Firebase
  • firestore.rules : Définit les règles de sécurité pour l'accès aux données
  • firestore.indexes.json : Configure les index pour optimiser les requêtes
  • .firebaserc : Spécifie le projet Firebase à utiliser

Scripts Utilitaires

Le dossier scripts/ contient des outils d'administration :

  • createAdmin.js : Script pour créer un nouvel administrateur
